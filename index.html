Keahlian Saya

HTML

Saya memahami struktur dasar HTML dan terbiasa menggunakan tag seperti HTML, head, body, div, img, a, dan form. Dalam membuat layout, saya lebih sering menggunakan elemen div untuk membagi bagian-bagian halaman karena sudah familiar dan mudah saya atur dengan CSS. Meskipun saya belum terlalu sering menggunakan elemen semantik seperti section, article, atau footer, saya terus belajar untuk memperbaiki struktur kode agar lebih rapi dan sesuai standar

CSS

Saya menggunakan CSS untuk mengatur tampilan website dan membuat layout yang rapi. Saya sudah paham cara menggunakan margin, padding, font, background, border, dan Flexbox untuk mengatur posisi elemen. Saya memang masih belajar tentang kombinasi warna, CSS Grid, dan desain yang sepenuhnya responsif, tetapi saya sudah mulai mencoba membuat tampilan website yang bisa menyesuaikan di berbagai ukuran layar. Saya juga suka bereksperimen dengan efek hover dan transition sederhana untuk menambah interaksi visual

JavaScript

Saya sedang berada di tahap belajar JavaScript dari dasar. Saya sudah memahami konsep seperti variabel, tipe data, operator, kondisi (if/else), dan fungsi. Meskipun saya belum sampai mempelajari manipulasi DOM dan membuat interaksi langsung dengan elemen HTML, saya tertarik untuk mempelajarinya lebih dalam. Saat ini saya fokus untuk menguatkan logika dasar JavaScript terlebih dahulu agar nantinya lebih mudah memahami fitur yang lebih kompleks
